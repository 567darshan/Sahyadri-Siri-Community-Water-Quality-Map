# Sahyadri Siri – Community Water Quality Map

## Overview
Sahyadri Siri is an Android application developed for monitoring and reporting water quality in natural streams and sacred springs of the Western Ghats (Sahyadri region).

The application allows local communities to report pollution, water clarity, flow condition, and stream health in real time.

## Features
- Community water quality reporting
- Water health monitoring map
- Pollution alert notifications
- Educational awareness section
- GPS-based reporting system

## Technologies Used
- Android Studio
- Java
- Firebase
- Google Maps API
- XML

## Project Objective
The objective of this project is to create a community-driven platform for protecting natural water resources and providing early warning systems for water pollution.

## Future Enhancements
